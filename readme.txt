=== ScanToolWP ===
Plugin para scannear el estado general del sitio y creación de libros.

== Description ==
Permite mostrar de forma ágil y rápida la información técnica de un sitio Web creado con WordPress en un solo lugar.

Conoce la información de tu sitio web como:
* Nombre del sitio.
* Url de instalación.
* Url de WordPress.
* Versión de WordPress.
* Listado de temas instalados.
* Listado de plugins instalados
* Número de páginas publicadas.
* Número de blogs publicados.

Además de la creación de tu propia galería de libros donde podrás tener la siguite información:

* Título del libro.
* Género.
* Author.
* Año de publicación.
* Imagen de portada.

== Installation ==

* Descarga el archivo ZIP con la versión más reciente del plugin.
* En el panel izquierdo, posiciona el puntero del mouse sobre el ícono de Plugins y selecciona Add New .
* Cargue el plugin ScanToolWP en su blog, actívelo y listo!.
* Para mostrar el listado de los libros solo debes agregar el siguiente shortcode [list_books].
